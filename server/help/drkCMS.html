1. Presentation

drkCMS est un gestionaire de contenu WEB tels que SPIP (mais largement simplifié), Ecrit en PHP et fonctionnant sur une base de données MySQL.
Il fonctionne sur un modèle Catégories / Rubriques / Articles.

1.1. Qu'est ce qu'un CMS ?

Un système de Gestion de Contenu (SGC) ou Content Management System (CMS) est un outil permettant de générer des site WEB structurés sans pour autant avoir besoin de grandes connaissances dans l'administration de site WEB. Il permet généralement de structurer son site web à travers des thèmes, et d'écrire des articles à l'intérieur de ces thèmes. Il nécessite génralement un Gestionnaire de Bases de données (souvent MySQL) et un langage permetant de faire fonctionner l'outil (souvent PHP). Le plus connus des CMS est SPIP : http://www.spip.net

1.2. Pourquoi drkCMS ?

drkCMS est lui aussi un Gestionnaire de Contenu, mais se veut tres simple et minimal :
  • peu d'élements constitue l'outil de gestion
  • seul l'administrateur (webmaster) peut structurer le site et ajouter des articles
  • la prise en main se veut rapide et efficace
il est cependant relativement puissant et modulaire :
  • on peut facilement l'interfacer avec toute autre ressource externe (grace aux rubriques externes)
  • on peut facilement y intégrer de nouvelle fonctionalités grace aux modules
  • il permet grace à son langage interne de générer des pages respectant le XHTML 1.0
  • on peut facilement paraéter la mise en forme graces aux feuilles de styles CSS 2

2. Installation

2.1. Configuration Requise

  • Un serveur HTTP supportant le PHP4
  • Le module PHP4
  • Un serveur de base de données MySQL

2.2. Configuration Recommendée

Il ne s'agit pas exactement de la configuration recommendée, mais plutôt de la configuration sur laquelle, j'ai développé drkCMS et qui donc est sure de le supporter :
  • Apache v0.1.2
  • PHP 4 v0.1.2
  • MySQL v3.2.125

2.3. Installation

Dans un premier temps vous devez décompresser l'archive contenant drkCMS, dans le répertoire contenant votre site WEB :
$ tar xzf drkcms-0.1.tar.gz ./
Ensuite vous devz créer votre base de données sous MySQL
$ mysql -uroot
mysql> create database drkcms;
Query OK, 1 row affected (0.00 sec)
Enfin executer sous MySQL le contenu du fichier sql/drkcms.sql
$ tar xzf drkcms-0.1.tar.gz ./
Vous pouvez alors suprrimer le répertoire sql
$ rm -rf sql/
Le site n'est cependant pas tout à fait pret, en effet il faut maintenant paramétrer drkCMS pour qu'il prenne en compte toute les informations relatives à votre site WEB.

2.4. Configuration

Afin de simplifier au maximum la configuration du votre site, tous les paramètres sont stockés dans le fichier inc/config.inc.php.

2.4.1. La configuration de base 

Voici les différents champs modifiables pour la configuration de votre site WEB :
 $CONFIG["dbhost"]="localhost"; Ceci indique quel est votre serveur MySQL
 $CONFIG["dbuser"]="root"; Ceci permet de spécifier l'utilisateur MySQL
 $CONFIG["dbpassword"]=""; Ceci permet de spécifier le Mot de passe de l'utilisateur MySQL
 $CONFIG["dbdatabase"]="drkcms"; Ceci est le nom de votre Base MySQL
 $CONFIG["titre"]="drkCMS - Gestionnaire de Contenue WEB"; Ceci est le titre de votre site web
 $CONFIG["keywords"]="drkCMS, Content, Manager, System, CMS, PHP, MySQL, free, software"; Ici on spécifie les mots clés pour les moteurs de recherches
 $CONFIG["theme"]="themes/default"; Ici on spécifie le répertoire du thème à utiliser
 $CONFIG["password"]="admin"; Ici on spécifie le mot de passe pour l'administration du site (ajout d'articles, de catégories, ...)
 $CONFIG["webmaster"]="Philippe Bousquet"; Ici on indique le Nom du Webmaster
  $CONFIG["adresse"]="28 cours du général De Gaulle, 33170 Gradignan"; L'adresse du Webmaster (Facultatif)
 $CONFIG["telephone"]="06.61.99.25.84"; Le numero de téléphone du Webmaster (Facultatif)
 $CONFIG["email"]="Darken33@free.fr"; L'email du Webmaster
 $CONFIG["navbar"]="Oui"; Ici on indique si l'on veut afficher la bare de navigation (Inutile si vous ne voulez qu'une seule Catégorie)
 $CONFIG["menu"]="Oui"; Ici on indique si l'on veut afficher le menu
 $CONFIG["date"]="Oui"; Ici on indique si l'on veut afficher la date de création des articles.

2.4.2. Les Modules de drkCMS

drkCMS permet l'ajout de modules spécifique dans le panel Gauche (celui du menu), drkCMS est fournit avec 4 Modules :
 $MODULES[0]="modules/admin.mod.php"; Module permetant la connection à l'administration du site (Ajout, modification d'Articles)
 $MODULES[1]="modules/visiteurs.mod.php"; Module affichant un compteur de visite (Spécifique à Free.fr)
 $MODULES[2]="modules/webmaster.mod.php"; Module affichant les infos du Webmaster
 $MODULES[3]="modules/hebergeur.mod.php"; Module affichant le logo de l'hebergeur
Vous pouvez bien entendu modifier les modules ou ajouter vos propres modules, pour cela il suffit de rajouter un élément dans le tableau :
 $MODULES[4]="modules/monmodule.php";

3. Utilisation de drkCMS

drkCMS se veut très simple, de ce fait il ne comprend que deux pages :
  • index.php : page principale, qui liste les articles d'une rubrique
  • article.php : cette page affiche le contenu d'un article

3.1. La page index.php

Cette page est le point d'entrée de drkCMS, elle est découpée en 7 parties :

La partie Entête
S'il existe un fichier "baniere.png" dans le répertoire du thème du site, drkCMS affichera celui-ci. Sinon il affcihera le titre du site contenu dans le fichier "inc/config.inc.php ".

La barre de navigation
Cette barre de navigation (si on demande de l'afficher) permet de sélectionner une catégorie.

Le menu
Ce menu permet d'afficher l'ensemble des rubriques contenus dans la catégorie courante.
Le fait de cliquer sur un élément du menu permet d'afficher la liste des articles de la rubrique concernée.

Les modules
Sous le menu on insère tous les modules listés dans le fichier "inc/config.inc.php".

Description de la catégorie
Si la catégorie en cours possède un texte de présentation celui ci est affcihé

La liste des articles
Cette parti affiche la liste des articles contenus dans la rubrique courante.
Si l'article possède un contenu lorsque l'on clique sur le titre de l'article, on se débranche vers la page "article.php".
Si l'article contient un texte d'introduction, celui est affiché.

Le pied de page
Le pied de page affiche des information sur drkCMS.

3.1. La page article.php

Cette page page permet d'afficher le contenu d'un article sélectionné à partir de la page index.php, elle est découpée en 6 parties :

La partie Entête
S'il existe un fichier "baniere.png" dans le répertoire du thème du site, drkCMS affichera celui-ci. Sinon il affcihera le titre du site contenu dans le fichier "inc/config.inc.php".

La barre de navigation
Cette barre de navigation (si on demande de l'afficher) permet de sélectionner une catégorie (on retourne sur la page index.php).

Le menu
Ce menu permet d'afficher l'ensemble des rubriques contenus dans la catégorie courante.
Le fait de cliquer sur un élément du menu permet d'afficher la liste des articles de la rubrique concernée (on retourne sur la page index.php).

Les modules
Sous le menu on insère tous les modules listés dans le fichier "inc/config.inc.php".

Le contenu de l'article
Dans cette partie on affcihe le contenu complet de l'article sélectionné.

Le pied de page
Le pied de page affiche des information sur drkCMS.


4. Administration

Le module administration inclus dans drkCSM, permet de se connecter au panneau d'administration du site. Pour cela il suffit de cliquer sur ce connecter, puis d'entrer le mot de passe définit dans le fichier "inc/config.inc.php" $CONFIG["password"].

4.1. Gérer ses Catégories

Une fois identifiés vous vous débrancher sur la liste des catégories (ce sont les titre proposés dans la barre de navigation du site) disponibles sur votre site et vous pouvez alors :
  • Ajouter une nouvelle catégorie
  • Editer une catégorie existante
  • Supprimer une catégorie (celle ci ne doit contenir aucune rubrique, ni aucun article). ATTENTION : aucune confirmation n'est demandé lorsque vous supprimez une catégorie
  • Gérer les rubriques contenues dans la catégorie

Une Catégorie est définit par :
  • Un nom (qui saffiche dans la barre de navigation)
  • Un descriptif (vous pouvez ici utiliser le langage interne pour agrémenter votre texte)
  • Un numero d'ordre (qui indique la position de la catégorie dans la barre de navigation)

4.2. Gérer ses Rubriques

Cette liste permet de gérer le menu de la catégorie courante, en effet vous pouvez :
  • Ajouter une nouvelle rubrique
  • Editer une rubrique existante
  • Supprimer une rubrique (celle ci ne doit contenir aucun article). ATTENTION : aucune confirmation n'est demandé lorsque vous supprimez une rubrique
  • Gérer les articles contenus dans la rubrique

Une Rubrique est définit par :
  • Un nom (qui saffiche dans le menu de la Catégorie)
  • Un lien externe (à renseigner si vous voulez pointer vers une ressource externe à drkCMS)
  • Un numero d'ordre (qui indique la position de la rubrique dans le menu)

4.3. Gérer ses Articles

Cette liste permet de gérer les articles contenus dans une rubrique, en effet vous pouvez :
  • Ajouter un nouvel article
  • Editer un article
  • Supprimer un article. ATTENTION : aucune confirmation n'est demandé lorsque vous supprimez un article
  • Prévisualiser un article

Un Article est définit par :
  • Un titre (qui saffiche dans la liste des articles)
  • Une Introduction (vous pouvez ici utiliser le langage interne pour agrémenter votre texte)
  • Un Contenu (vous pouvez ici utiliser le langage interne pour agrémenter votre texte)
  • Un numero d'ordre (qui indique la position de l'article dans la liste)
  • Un Etat Visible/Invisible "Invisible" par défaut(Permet de ne pas afficher un article en cours d'edition)
